# OpenGLCar3D

**OpenGLCar3D** is an interactive 3D car racing game developed using C++ and OpenGL (GLUT).  
This project features real-time rendering, lane-based movement, collision detection, a countdown start system, score tracking, and timed gameplay mechanics.

---

## Features

- Real-time 3D rendering using OpenGL  
- Lane-based car movement controls  
- Collision detection and response  
- Countdown timer before race start  
- Score tracking and time limits for gameplay  

---

## Getting Started

### Prerequisites

- C++ compiler supporting C++11 or later  
- OpenGL and GLUT libraries installed on your system  

### Running the Game

1. Clone the repository or download the source files.  
2. Compile the source code (`main.cpp`) using your preferred IDE or command line.  
3. Run the resulting executable to start the game.
